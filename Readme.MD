# 🥤 Cadeya Smoothies

Application web de génération de smoothies santé à base d'ingrédients locaux d'Afrique de l'Ouest.

**Version Pro 1.0** · Cahier des Charges v5.0 · © 2026 Cadeya Smoothies

## Fonctionnalités

- 96 mélanges validés · 6 catégories
- Tirage Fisher-Yates · Anti-doublon session
- Compteur dynamique 1–20 personnes
- Export PDF professionnel
- Espace Admin sécurisé
- Tarification PayDunya (Phase 2)
